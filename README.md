# Comandos Úteis 

## Git

```
# Inicializa o Git em uma nova pasta ou projeto
git init 
```
```
# Configura seu nome e e-mail globalmente no git
git config --global user.name "Tulio Koogi"
git config --global user.email tulio-koogi@hotmail.com

```
```
# Lista a configurações do git
git config --list

```
```
# Comandos para commitar
git status
git add .
git commit -m "Mensagem do commit"
```
```
# Publicar todos os commits (Mandar para o repositório na web)
git push
```
